Username Generator

Build a custom format using tokens, then generate as many usernames as you need instantly.

Format
Word type
adj noun n1 n2 n3 n4
Modifier (optional, stackable)
upper lower cap leet
Syntax %[modifier:]type%
Word types
adj → random adjective e.g. silent, lunar, crisp
noun → random noun e.g. prism, wolf, tide
n1 · n2 · n3 · n4 → random number with 1–4 digits
Modifiers
upper → ALL CAPS  %upper:adj% → SILENT
lower → all lowercase  %lower:adj% → silent
cap → Capitalised first letter  %cap:noun% → Prism
leet → 1–2 leet substitutions  %leet:adj% → s1lent
Stacking modifiers
%leet:upper:adj% → leet first, then uppercase  → S1L3NT
%cap:noun%_%n2% → capitalised noun + 2-digit number  → Prism_47
